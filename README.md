## Collaboration and Workflow
- This repository uses GitHub collaboration access
- Each feature or bug fix must be developed in a separate branch
- Direct pushes to the `main` branch are not allowed
- All pull requests must be reviewed and approved by the Team Lead
- Follow clean, readable, and well-documented code practices

## Tech Stack
- Python
- Django
- Django REST Framework (DRF)

- Develop RESTful APIs
- Implement business logic
- Handle authentication and authorization
- Validate input data and handle errors
- Maintain backend code quality
- Provide API contracts to the frontend team
